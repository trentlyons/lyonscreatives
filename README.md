# GitHub Pages Starter Site

This is a minimal static website for testing GitHub Pages.

## Files included

- `index.html`
- `style.css`
- `script.js`
- `404.html`

## How to publish

1. Create a GitHub repository.
2. Upload all files from this folder into the repository root.
3. In the repository, go to **Settings** → **Pages**.
4. Under **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ (root)**
5. Save and wait for the site to publish.

## Best repo name for simplest setup

If your GitHub username is `yourusername`, name the repository:

`yourusername.github.io`

Then your site URL should become:

`https://yourusername.github.io`

## What to test after deployment

- The homepage loads
- Styling is applied
- Navigation scrolls
- The "Test JavaScript" button works
